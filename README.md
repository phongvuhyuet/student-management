## Main features: 
* 二要素認証、ログインセッション管理
* 指導教官の管理下に新しい学生を追加します
* 各学期の学生の学習結果を要約します
* タスクの割り当てと受信、進行状況の更新の機能があります。
* アドバイザーと学生のチャット機能があります
* 授業料不足で中退する恐れのある生徒をまとめる機能があります
